## Общая картина

Эта страница — твой **чеклист по направлениям**, чтобы видеть прогресс.

## 1. Linux и базовые навыки

- Уверенное владение Linux:
  - файловая система
  - пакеты
  - процессы
  - права
- Работа с CLI
- Скрипты (Bash / Python)
- Базовое понимание сетей:
  - TCP/IP
  - DNS
  - HTTP/HTTPS
  - firewall

## 2. Git

- Основные команды (init, clone, add, commit, status, log)
- Ветвление (branch)
- merge / rebase
- Работа с GitHub / GitLab (remotes, PR/MR)

## 3. Docker

- Образы (image)
- Контейнеры (container)
- Dockerfile
- volumes
- networks

## 4. CI/CD

- Понимание концепции CI/CD
- GitLab CI:
  - пайплайн: build → test → deploy
  - основы `.gitlab-ci.yml`

## 5. Kubernetes и оркестрация

- Kubernetes:
  - Pods
  - Deployments
  - Services
  - Ingress
  - ConfigMaps / Secrets
- Helm — пакетный менеджер
- `kubectl` и манифесты
- Локальный кластер:
  - kind
  - minikube
  - k3s

## 6. IaC (Infrastructure as Code)

- Terraform — стандарт де-факто
- Ansible — конфигурация серверов

## 7. Облака

- Создание:
  - виртуальных машин
  - сетей
  - балансировщиков
  - хранилищ
- Работа через:
  - UI
  - CLI / SDK
  - Terraform
- Yandex Cloud (мини-курс «облачный инженер» и аналоги)

## 8. Мониторинг и логи

- Prometheus + Grafana (метрики)
- Loki / ELK / OpenSearch (логи)
- Alertmanager — алерты

## 9. Безопасность

- Минимизация привилегий
- Secrets management:
  - Vault
  - Kubernetes Secrets
- Сетевые политики (NetworkPolicy в K8s)
- SELinux / AppArmor (базово)
