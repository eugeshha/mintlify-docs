## О чём была лекция

- Что такое Kubernetes и зачем он нужен

## Базовые понятия

- **Cluster** — набор узлов (nodes), управляемых control plane
- **Node** — рабочая машина (VM или физический сервер), где крутятся поды
- **Pod** — минимальная единица деплоя (один или несколько контейнеров)
- **Control Plane** (управляющая плоскость):
  - API server
  - etcd
  - scheduler
  - controller manager
- **Worker Node**:
  - kubelet
  - kube-proxy
  - контейнерный runtime (containerd / CRI-O)
