## Назначение probes

- **Что это**: механизмы периодической проверки состояния контейнеров.
- **Для чего**: определить, готово ли приложение обслуживать трафик и нужно ли перезапустить контейнер.

## Типы probes

- **Liveness probe**: отвечает за «живость» контейнера; при неуспехе Kubernetes перезапускает Pod.
- **Readiness probe**: показывает готовность принимать запросы; при неуспехе Pod временно исключается из Service.
