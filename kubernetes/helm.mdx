## Назначение Helm

- **Что это**: инструмент для упаковки, версионирования и установки Kubernetes‑манифестов в виде чартов.
- **Для чего**: упростить управление сложными наборами ресурсов и параметров.

## Ключевые элементы

- **Chart**: пакет Kubernetes‑ресурсов с шаблонами и значениями по умолчанию.
- **Values**: файл с параметрами конфигурации для конкретной установки.
- **Release**: установленный экземпляр чарта в конкретном namespace.

## Практическое применение

- **Повторяемые установки**: развёртывание одинаковых стеков в разных окружениях.
- **Управление настройками**: изменение параметров через values, без переписывания манифестов.
- **Обновления**: контролируемые апдейты и откаты релизов.
