## Роль облаков

- **Что это**: платформа, предоставляющая вычислительные ресурсы по запросу.
- **Для чего**: быстро поднимать и масштабировать инфраструктуру без управления железом.

## Основные ресурсы

- **Виртуальные машины (VM)**: базовые вычислительные единицы для приложений.
- **Сети и подсети**: изоляция трафика и маршрутизация.
- **Балансировщики**: распределение входящего трафика между экземплярами.
- **Хранилища**: диски, объектные хранилища, базы данных как сервис.

## Практическое применение

- **Создание окружений**: dev/stage/prod с однотипными ресурсами.
- **Интеграция с Terraform**: управление ресурсами как кодом.
- **Локальные аналоги**: Yandex Cloud и другие провайдеры с похожими примитивами.
